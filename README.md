# BMA Simulator – Arduino Mega 2560 / Wokwi

Dieses Repository ist für den direkten Import in Wokwi vorbereitet.

[![Open in Wokwi](https://wokwi.com/assets/images/wokwi-badge.svg)](https://wokwi.com/projects/new/arduino-mega?template=https://github.com/huschteklumbe/BMA-Simulator)

## Nutzung

1. Dieses Projekt als neues GitHub-Repository hochladen.
2. In der README den Link oben anpassen:
   - `DEIN-USER` durch deinen GitHub-Benutzernamen ersetzen
   - `DEIN-REPO` durch den Repository-Namen ersetzen
3. Auf **Open in Wokwi** klicken.
4. Simulation starten.

## Enthaltene Dateien

- `sketch.ino` – Arduino Mega Programm
- `diagram.json` – Wokwi Verdrahtung
- `libraries.txt` – verwendete Bibliotheken
- `README.md` – diese Anleitung

## Hinweise

Die Ausgänge für Alarmtröte und Blitzleuchte sind in Wokwi als LEDs dargestellt.
In echter Hardware müssen diese Ausgänge über Relais oder MOSFET-Treiber geschaltet werden.
